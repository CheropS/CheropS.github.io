The Maze: SDL2 GAME

About the Game

This is a game created by SDL2 that allows players to walk around a maze.

A player navigates through the maze by using the arrows on their keyboards

Once a player hits an obstacle, they are supposed to figure out how to navigate around it.

How to Download

The game has been created for school purposes only.

A person heads over to my github page via the link git clone

They then clone the game and follow instructions as directed on the github page

How to Play

After cloning, a player runs the command on the terminal ./maze

Another window will pop up with the game

Navigate going forward by pressing down arrow key

Navigate going back by pressing up arrow key

Navigate going sideways by pressing either left or right arrow keys
